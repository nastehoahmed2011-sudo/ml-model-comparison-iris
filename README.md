# Machine Learning Model Comparison (Iris Dataset)

## Overview
This project compares multiple machine learning classification models using the Iris dataset in R. The goal is to evaluate model performance using cross-validation and identify the most accurate approach.

## Models Used
- Linear Discriminant Analysis (LDA)
- Support Vector Machine (SVM)
- Random Forest

## Methodology
- 80/20 train-test split
- 10-fold cross-validation
- Accuracy used as primary evaluation metric

## Key Result
Random Forest achieved the highest classification accuracy among tested models.

## Tools
- R
- caret
- e1071
- randomForest
- MASS
